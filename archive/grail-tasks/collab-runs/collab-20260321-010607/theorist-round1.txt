## Theorem 1 (Fiber Threshold Lemma)

**Proof.** Within F_c, the projections ╧А(v) = v mod ker(w) give |F_c| elements in F_p^r.

*Claim:* these projections are pairwise distinct. If ╧А(v) = ╧А(v') for v, v' тИИ F_c, then v тИТ v' тИИ ker(w) with w(v тИТ v') = c тИТ c = 0. But also v тИТ v' has zero projection, so v = v'. тЬУ

Each projection appears with multiplicity pтИТ1 in the sequence. Total length: |F_c|(pтИТ1) тЙе (╬▒(r)+2)(pтИТ1).

By inductive hypothesis: s(C_p^r) = ╬╖(C_p^r) + p тИТ 1 тЙд ╬▒(r)(pтИТ1) + 1 + p тИТ 1 = (╬▒(r)+1)(pтИТ1) + 1.

Since (╬▒(r)+2)(pтИТ1) = (╬▒(r)+1)(pтИТ1) + (pтИТ1) тЙе (╬▒(r)+1)(pтИТ1) + 1 = s(C_p^r) for p тЙе 3, the s-constant guarantees a zero-sum subsequence of length exactly p.

This subsequence corresponds to coefficients s_i тИИ {0,...,pтИТ1} for each v тИИ F_c with ╬гs_i = p and ╬гs_i ╧А(v_i) = 0. The first coordinate: c ┬╖ ╬гs_i = cp тЙб 0 mod p. So ╬гs_i v_i = 0 with weight p тЙд p. тИО

## Theorem 2 (Dimensional Fiber Threshold)

**Statement.** Under the same setup, let F_c (c тЙа 0) be a non-kernel fiber whose projections span a subspace of dimension d (where 1 тЙд d тЙд r). If |F_c| тЙе ╬▒(d)+2, then S contains a weighted zero-sum of weight exactly p using only vectors from F_c.

Here ╬▒(1) = 1, ╬▒(2) = 3 (from EGZ/Kemnitz), ╬▒(k) = 9┬╖2^{kтИТ3}тИТ1 for k тЙе 3.

**Proof.** Identical to Theorem 1, but applying s(C_p^d) instead of s(C_p^r). The projections lie in a d-dimensional subspace V_d тЙЕ F_p^d. The s-constant s(C_p^d) тЙд (╬▒(d)+1)(pтИТ1)+1. The argument goes through verbatim. тИО

**Corollary (ZSF Fiber Size Bounds by Dimension).** In a ZSF set, every non-kernel fiber F_c with projection dimension d satisfies:
- d = 1 (projections on a line): |F_c| тЙд 2

## Theorem 3 (Complete Inductive Reduction)

**Statement.** For r тЙе 3 and p тЙе 5 prime, assume ╬╖(C_p^r) = ╬▒(r)(pтИТ1)+1 (both bounds). Then ╬╖(C_p^{r+1}) = (2╬▒(r)+1)(pтИТ1)+1 if and only if the following condition holds:

**(MF_r):** Every set S of 2╬▒(r)+2 nonzero non-proportional vectors in F_p^{r+1} that spans F_p^{r+1} and satisfies, for *every* linear form w, that at least 3 fibers are nonempty тАФ must contain a weighted zero-sum of weight тЙд p.

**Proof.**

(тЯ╣) Trivially, any such S has 2╬▒(r)+2 vectors, so the ╬╖ bound gives a zero-sum.

(тЯ╕) Given S with |S| = 2╬▒(r)+2.

**Case A:** S lies in a subspace V of dimension d тЙд r. Then |S| = 2╬▒(r)+2 > ╬▒(d)+1 (since ╬▒ is strictly increasing for d тЙе 1). By ╬╖(C_p^d) at rank d, the sequence of length |S|(pтИТ1) тЙе ╬╖(C_p^d), so a zero-sum exists. тЬУ

**Case B:** S spans F_p^{r+1} and some linear form w gives тЙд 2 nonempty fibers. Apply the Two-Fiber Bound (proved). тЬУ

**Case C:** S spans F_p^{r+1} and every w gives тЙе 3 fibers. Apply (MF_r). тЬУ

*Tightening Case B:* With 2 fibers F_{c_1}, F_{c_2} and total 2╬▒(r)+2: either both are non-kernel (change basis to make one the kernel, apply Two-Fiber Bound), or one is the kernel F_0 and one is F_c. Then |F_0| + |F_c| = 2╬▒(r)+2. If |F_0| тЙе ╬▒(r)+1: ╬╖ in kernel. If |F_c| тЙе ╬▒(r)+2: Theorem 1. Since |F_0| + |F_c| = 2╬▒(r)+2, we can't have both |F_0| тЙд ╬▒(r) and |F_c| тЙд ╬▒(r)+1 (that gives тЙд 2╬▒(r)+1 < 2╬▒(r)+2). тЬУ тИО

**Key insight:** Condition (MF_r) is about **3-spread sets** тАФ sets not contained in any union of 2 parallel hyperplanes. The extremal construction (2╬▒(r)+1 vectors) IS contained in 2 parallel hyperplanes (fibers 0 and 1 under x_1). So 3-spread sets are **structurally distinct** from the extremal construction. This is what must be exploited.

## The Precise Obstruction: What Remains

Combining Theorems 1тАУ3 and Propositions 4тАУ5, the problem reduces to:

**Condition (MF_r):** For a 3-spread set S of 2╬▒(r)+2 vectors in F_p^{r+1}, prove a weighted zero-sum of weight тЙд p exists.

The available tools for attacking (MF_r):

1. **Cross-fiber partial sums** (Prop 4): for any pair of non-kernel fibers, we get partial sums with budget for kernel cancellation.

2. **Dimensional thresholds** (Thm 2): fibers with low-dimensional projections must be tiny, so most vectors are in "full-dimensional" fibers.

3. **Rank-3 subroutine** (collinear triples): any non-degenerate collinear triple gives a zero-sum. The only obstacle is pairwise-degenerate triples.

4. **The 3-spread constraint**: S is not in any thick hyperplane, meaning for every projection, vectors are genuinely spread across тЙе 3 fibers.

**Conjecture (Multi-Fiber Representation).** For a 3-spread ZSF set S, the 3-spread condition forces enough cross-fiber interaction that SOME combination of Prop 4 (cross-fiber partial sums) + kernel representation yields a zero-sum. Specifically:

*For any 3-spread set of 2╬▒(r)+2 vectors, there exist two non-kernel fibers c_1, c_2 (under some linear form w), vectors v_1 тИИ F_{c_1}, v_2 тИИ F_{c_2}, a coefficient s_1 тИИ {1,...,pтИТ1}, and kernel vectors u_1, ..., u_m тИИ F_0, such that the residual h = s_1 ╧А(v_1) + тЯитИТs_1 dтЯй ╧А(v_2) is representable as тИТ╬г t_j u_j with weight ╬гt_j тЙд тЯиs_1 dтЯй тИТ s_1.*

## Strategic Assessment

The **most promising path** to proving (MF_r) is not direct cross-fiber analysis, but rather:

**The CW-Polynomial Method (Grynkiewicz 2023 + Reiher technique):**

Reiher proved s(C_p^2) = 4pтИТ3 by:
1. Counting zero-sum subsequences of length p via Chevalley-Warning congruences
2. Showing the count N_p тЙб 0 mod p
3. Using combinatorial structure to bound N_p away from 0

For rank r+1, CW gives ~(r+1) congruences. The rank-3 theorem provides **additional structural constraints** (within each fiber). Nobody has combined these. The key paper is Grynkiewicz, "A Generalization of the Chevalley-Warning and Ax-Katz Theorems," *Combinatorica* 43(6), 2023.

The approach would be: apply generalized CW to the system defining weight-p zero-sums in C_p^{r+1}, then use the rank-r ╬╖ bound as a structural input to show the count must be positive.
