# THEORIST REPORT: Round 2

## CRITICAL CORRECTION: The Cap Assumption is Wrong

**The most important finding this round is a fundamental error in prior reasoning (including my own Round 1 report).**

**Claimed (Round 1 and earlier sessions):** A ZSF set must be a "cap" — no 3 vectors can be collinearly dependent (linearly dependent), because any collinear triple can be rescaled to a zero-sum of weight p.

**This is FALSE.** A collinear triple a₁v₁ + a₂v₂ + a₃v₃ = 0 does NOT always yield a zero-sum of weight ≤ p after rescaling.

**Counterexample.** At p = 5, consider the dependency with normalized coefficients (1, 1, 4). Weight at each μ ∈ F₅*:

| μ | ⟨μ·1⟩ | ⟨μ·1⟩ | ⟨μ·4⟩ | Weight |
|---|--------|--------|--------|--------|
| 1 | 1 | 1 | 4 | 6 |
| 2 | 2 | 2 | 3 | 7 |
| 3 | 3 | 3 | 2 | 8 |
| 4 | 4 | 4 | 1 | 9 |

Minimum weight = 6 > p = 5. **No rescaling achieves weight ≤ p.**

## Theorem 1: The Weight Dichotomy for Triples (PROVED)

**Theorem.** Let (a₁ : a₂ : a₃) ∈ PG(2, p) be the projective dependency of a collinear triple. Define w(μ) = ⟨μa₁⟩ + ⟨μa₂⟩ + ⟨μa₃⟩. Then:

(i) **Pairing identity:** w(μ) + w(p - μ) = 3p for all μ ∈ F_p*.

(ii) **Good iff extreme:** min_μ w(μ) ≤ p ⟺ max_μ w(μ) ≥ 2p.

(iii) **The "moderate coefficient" obstruction:** min_μ w(μ) > p ⟺ w(μ) ∈ {p+1, ..., 2p-1} for ALL μ.

**Proof.**

(i) w(μ) + w(p-μ) = Σᵢ [⟨μaᵢ⟩ + ⟨(p-μ)aᵢ⟩] = Σᵢ [⟨μaᵢ⟩ + ⟨-μaᵢ⟩] = Σᵢ p = 3p. Here we use that ⟨x⟩ + ⟨-x⟩ = p for any x ∈ F_p*.

(ii) From (i): min_μ w(μ) = 3p - max_μ w(μ). So min ≤ p ⟺ max ≥ 2p.

(iii) If min > p, then all w(μ) > p. By (i), w(p-μ) = 3p - w(μ) < 3p - p = 2p. So all w(μ) < 2p. Combined: w(μ) ∈ {p+1, ..., 2p-1} for all μ. ∎

**Corollary (Classification at p = 5).** A normalized dependency (1, a, b) with a ≤ b is:
- **Good** (weight ≤ 5 achievable): (1,1,1), (1,1,2), (1,1,3), (1,3,3)
- **Bad** (weight > 5 for all μ): (1,1,4), (1,2,3), (1,2,4), (1,3,4), (1,4,4)

That's 4 good types vs 5 bad types. **More than half of all dependency types are "bad" at p = 5.**

## Theorem 2: The Complementary Pair Obstruction (CPO) — PROVED

**Theorem.** A support-k dependency (a₁, ..., a_k) with k even, whose coefficients consist of k/2 complementary pairs {a, p-a}, has constant weight kp/2 for all μ ∈ F_p*.

**Proof.** For each pair {a, p-a}: ⟨μa⟩ + ⟨μ(p-a)⟩ = ⟨μa⟩ + ⟨-μa⟩ = p. Total: (k/2) · p = kp/2. Independent of μ. ∎

**Corollary.** Such a dependency gives weight ≤ p only if k ≤ 2. For k = 4: weight = 2p > p. For k = 6: weight = 3p > p.

**Theorem (Geometric Characterization of CPO).** Four vectors v₁, v₂, v₃, v₄ have a dependency with complementary pair coefficients iff v₁ - v₂ is proportional to v₃ - v₄ (under some pairing of the four vectors).

**Proof.** If the dependency is a(v₁) + (p-a)(v₂) + b(v₃) + (p-b)(v₄) = 0, this gives a(v₁ - v₂) + b(v₃ - v₄) = 0, i.e., v₁ - v₂ = -(b/a)(v₃ - v₄). Proportional. ∎

**This means CPO ↔ "parallel differences" ↔ projective parallelogram.**

## Theorem 3: Revised ZSF Structure (PROVED)

**Theorem (ZSF = Avoiding Good Dependencies).** A set S ⊂ F_p^r \ {0} of non-proportional vectors, each with multiplicity p-1, is zero-sum free iff it avoids all "good" dependencies — dependencies (a₁, ..., a_k) over subsets of S with min_μ Σ⟨μaᵢ⟩ ≤ p.

**Key consequences of the corrected theory:**

1. **ZSF sets CAN contain collinear triples** (linearly dependent triples), provided all such triples have "bad" coefficients (min weight > p for all rescalings).

2. **ZSF sets CAN contain 4-vector dependencies** of complementary-pair type (weight always 2p).

3. **The "cap" condition (no collinear triple) is STRICTLY STRONGER than ZSF.** The true constraint on ZSF sets is weaker: avoid "good" dependencies only.

4. **This resolves the |A₈| = 8 vs cap-size-6 paradox at p = 5.** The 8-element ZSF set A₈ in F₅³ has collinear triples, but only "bad" ones. It is NOT a cap.

## Theorem 4: When Support-3 Gives Weight p (PROVED)

**Theorem.** For a dependency (a₁ : a₂ : a₃) with Σaᵢ = p (integer sum), the weight at μ = 1 is exactly p.

**Proof.** w(1) = a₁ + a₂ + a₃ = p. ∎

**More generally:** a dependency has w(μ) = p iff the integer sum of the rescaled representatives equals p, i.e., ⟨μa₁⟩ + ⟨μa₂⟩ + ⟨μa₃⟩ = p. This requires no "wrap-around" in the mod-p reduction except for the total summing to p.

**Theorem (Complementary Triple Weight).** For a dependency of the form (1, a, p-1-a) with a ∈ {1, ..., p-2}:

w(μ) ∈ {p, 2p} for all μ. Specifically, w(μ) = p when μ(1+a) < p (no wrap), and w(μ) = 2p when μ(1+a) ≥ p (wrap occurs).

In particular, w(1) = 1 + a + (p-1-a) = p. So these are always "good."

**Proof.** Write a₃ = p - 1 - a = -(1+a) mod p. Then ⟨μa₃⟩ = p - ⟨μ(1+a)⟩. The sum w(μ) = ⟨μ⟩ + ⟨μa⟩ + p - ⟨μ(1+a)⟩. Since ⟨μ⟩ + ⟨μa⟩ = ⟨μ(1+a)⟩ + εp where ε ∈ {0,1}: w(μ) = p + εp ∈ {p, 2p}. ∎

## Theorem 6: The CW Solution Count (PROVED, extends R1 Thm)

**Theorem (Chevalley-Warning + Ax-Katz).** Let S = {v₁, ..., v_n} be non-proportional vectors in F_p^{r+1}. The number N of solutions to {Σsᵢvᵢ = 0, Σsᵢ ≡ 0 mod p} with sᵢ ∈ F_p satisfies:

p^{n-r-2} | N, where the system has r+2 equations of degree 1 in n variables.

For n = 2α(r)+2, r ≥ 3: the exponent is n - r - 2 = 2α(r) - r ≥ 2·8 - 3 = 13 (at r = 3).

**The obstacle:** These N ≥ p^{2α(r)-r} solutions split among weight classes {0, p, 2p, ...}. CW/Ax-Katz cannot distinguish weight classes. Weight-0 accounts for exactly 1 solution (the trivial solution). The remaining p^{2α(r)-r} - 1 solutions have weight ≥ p, but we need at least one with weight EXACTLY p.

**Why direct CW fails:** The weight-exactly-p constraint is an INTEGER constraint (not a mod-p condition), so it cannot be encoded in the polynomial framework. Encoding it as a polynomial increases the total degree by (p-1), potentially violating the CW hypothesis.


The "good-dep-free" condition is WEAKER than "cap." This makes the problem HARDER: the extremal ZSF sets are larger than the largest caps (8 > 6 at p=5 in PG(2,5)), and the constraint we must violate is more permissive.
