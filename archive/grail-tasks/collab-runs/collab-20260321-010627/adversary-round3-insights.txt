# ADVERSARY AGENT: KEY INSIGHTS FOR ROUND 3

## Insight 1: Shadow Saturation Proof Has a Critical Gap
The core argument that η(C_p^r) = α(r)(p-1)+1 forces Σ_{p-1}(A) = F_p^r \ {0} relies on the claim that A^{p-1} ∪ {-v} must contain a zero-sum using -v exactly once. But this only holds if η is tight *and* A^{p-1} truly blocks all zero-sums of weight ≤ p—the latter requires verification that α(r)(p-1) < η-1, which hasn't been rigorously confirmed for r ≥ 2.

## Insight 2: Same-Tail Swap Counterexamples Undermine Uniqueness Claims  
The Universal Swap Theorem correctly explains why 7 single-swap variants of doubled+e1 at p=5 remain strongly ZSF (swaps preserve the property when fiber membership unchanged), but this directly contradicts any claim of rigidity. If α(4) truly admits non-equivalent strongly ZSF constructions, the inductive uniqueness argument—crucial for bounding α(r+1)—collapses at the base case.

## Insight 3: Zero-Surplus Bottleneck is Avoidable via Weight Redistribution
The bottleneck assumes fiber distributions stay (α(r-1), α(r-1), 2) or worse, but computations at p=5,7 show the doubled construction *escapes* this by distributing weight across *all* fibers with surplus > 0. The inductive step may fail not because α(r+1) is bounded, but because the prescribed-weight sumset problem forces a different architecture that the current weight-dichotomy argument doesn't capture.
