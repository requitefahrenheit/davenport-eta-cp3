# THEORIST ROUND 3: TWO NEW THEOREMS + REFORMULATED GAP

## THEOREM 1: s-CONSTANT FIBER LIFT (NEW)

**Statement.** Let r ≥ 3, p ≥ 5. If a nonzero fiber F_c (c ≠ 0) of a linear form w: F_p^{r+1} → F_p contains ≥ α(r)+2 non-proportional nonzero vectors from a set S, then S contains a zero-sum of weight exactly p.

**Proof.**

The vectors in F_c have the form (c, a_j) where a_j = π(v_j) ∈ F_p^r are the tails. Since the vectors are non-proportional and lie in the same nonzero fiber, the tails are pairwise distinct (proof: if a_j = a_k, then v_j = (c, a_j) and v_k = (c, a_k) = (c, a_j) = v_j, contradicting distinctness; proportionality v_k = λv_j forces λc = c hence λ = 1 hence v_k = v_j, contradiction).

So we have n_c = |F_c| ≥ α+2 distinct tails in F_p^r. Each vector appears with multiplicity p-1 in the copies pool. The tail copies pool has n_c(p-1) ≥ (α+2)(p-1) elements.

Now recall s(C_p^r) = η(C_p^r) + p - 1 = α(p-1) + 1 + p - 1 = α(p-1) + p.

Compute: (α+2)(p-1) = α(p-1) + 2(p-1) = [α(p-1) + p] + (p-2) = s(C_p^r) + (p-2).

Since p ≥ 5: (α+2)(p-1) ≥ s(C_p^r) + 3 > s(C_p^r).

By definition of s(C_p^r): every sequence of ≥ s(C_p^r) elements in C_p^r contains a zero-sum subsequence of length exactly p. So the tail copies pool contains a zero-sum Σ s_j a_j = 0 with Σ s_j = p (length exactly p, all s_j ∈ {1,...,p-1}).

Lifting to F_p^{r+1}: Σ s_j (c, a_j) = (c · Σ s_j, Σ s_j a_j) = (c · p, 0) = (0, 0) in F_p^{r+1}.

This is a zero-sum in F_p^{r+1} of weight exactly p. **QED.**

**Remark.** This argument is SHARP: the doubled+e1 construction places exactly α+1 vectors in the nonzero fiber, giving (α+1)(p-1) = s(C_p^r) - 1 copies — exactly one below the s threshold. The zero-tail vector e₁ acts as a placeholder that occupies one slot without contributing a useful tail, keeping the pool sub-critical.

## THEOREM 2: TWO-FIBER INDUCTION (NEW)

**Statement.** For r ≥ 3 and p ≥ 5: given 2α(r)+2 non-proportional nonzero vectors in F_p^{r+1}, if there exists a nonzero linear form w such that S has at most 2 nonempty fibers under w, then S contains a zero-sum of weight ≤ p.

**Proof.** Let F_0 = S ∩ ker(w), F_c = S ∩ w^{-1}(c) for some c ≠ 0. All vectors of S lie in F_0 ∪ F_c. Set n_0 = |F_0|, n_c = |F_c|, so n_0 + n_c = 2α + 2.

**Case A: n_0 ≥ α+1.**
F_0 consists of n_0 ≥ α+1 non-proportional nonzero vectors in ker(w) ≅ F_p^r. The copies pool has n_0(p-1) ≥ (α+1)(p-1) = α(p-1) + (p-1) ≥ α(p-1) + 1 = η(C_p^r).

By the definition of η: there exists a nonempty zero-sum of weight ≤ p within F_0. Since F_0 ⊂ ker(w) ⊂ F_p^{r+1}, this is a zero-sum in F_p^{r+1} with the same weight.

**Case B: n_0 ≤ α.**
Then n_c = 2α + 2 - n_0 ≥ α + 2. By Theorem 1 (s-Constant Fiber Lift): S contains a zero-sum of weight p.

Since Cases A and B are exhaustive (n_0 ≥ α+1 or n_0 ≤ α): **QED.**

**Corollary.** *The Grail η(C_p^{r+1}) = (2α(r)+1)(p-1)+1 holds for all sets admitting a 2-fiber partition.*

## STRUCTURAL ANALYSIS OF α-DISPERSED SETS

**Lemma (Multi-Fiber Forcing).** Any α-dispersed set of 2α+2 vectors has ≥ 3 nonempty fibers under EVERY linear form.

*Proof.* If only 2 fibers {F_0, F_c}: n_0 + n_c = 2α+2 with n_0 ≤ α ⟹ n_c ≥ α+2, violating the α-dispersal condition |F_c| ≤ α+1. Contradiction. **QED.**

**Lemma (Cross-Fiber Patterns Always Exist).** For 3+ nonempty fibers with total weight p, there always exist cross-fiber weight-p partition patterns (w_0, w_1, ...) with w_i > 0 for ≥ 2 nonzero fibers and Σ c_i w_i ≡ 0 mod p.

*Proof.* (Already proved by Adversary.) For fibers {0, c₁, c₂}: take w₀ = p - w₁ - w₂, and solve c₁w₁ + c₂w₂ ≡ 0 mod p. This has solutions with w₁, w₂ ≥ 1 since c₂ is invertible: w₁ = c₂⁻¹(p-c₁)... more precisely, we can verify for any 3 fibers, solutions exist by direct construction. **QED.**

**Key Structural Insight.** The α-dispersed condition creates a PARADOX:
- Every linear form gives ≥ 3 fibers (by Multi-Fiber Forcing)
- 3+ fibers always have cross-fiber weight-p patterns (by Adversary's theorem)
- Cross-fiber patterns + many vectors = many potential zero-sums
- Yet we need to show at least ONE is realized

## THEOREM 4: DISPERSED SETS HAVE ABUNDANT TAIL ZERO-SUMS (NEW)

**Statement.** For any α-dispersed set S of 2α+2 non-proportional nonzero vectors in F_p^{r+1} and any linear form w, the tail copies pool contains at least (α+2)(p-1) - 1 > η(C_p^r) elements, hence the tail space F_p^r admits a zero-sum of tails of weight ≤ p.

*Proof.* The 2α+2 vectors project to ≤ 2α+2 tails in F_p^r, each with multiplicity p-1 in the copies pool. Total: (2α+2)(p-1). Among the tails, at most 1 is the zero vector (projective uniqueness of the direction e₁). The nonzero tails contribute ≥ (2α+1)(p-1) copies to the pool.

Since (2α+1)(p-1) ≥ α(p-1)+1 = η(C_p^r) for α ≥ 1 and p ≥ 3: the pool exceeds η. By definition: a zero-sum of tails exists with weight ≤ p. **QED.**

**The remaining question:** among the many tail zero-sums, can we always LIFT one to F_p^{r+1} by satisfying the fiber constraint Σ s_j f_j ≡ 0 mod p?

## THEOREM 5: SPLIT FLEXIBILITY FOR DOUBLED TAILS (NEW)

**Statement.** Given a tail zero-sum Σ s_j a_j = 0 of weight W = Σ s_j ≤ p, suppose K of the participating tails are "doubled" (i.e., S contains two vectors with that tail, in different fibers). Let W_d = Σ_{doubled} s_j be the total weight on doubled tails. Then the set of achievable first-coordinate sums (fiber sums) has size ≥ min(p, W_d + 1).

*Proof.* For a doubled tail a_j with two vectors (f_j, a_j) and (f'_j, a_j), f_j ≠ f'_j: the tail coefficient s_j can be split as s_j^{(1)} + s_j^{(2)} = s_j with s_j^{(1)} ∈ {0,...,min(s_j, p-1)}, s_j^{(2)} = s_j - s_j^{(1)} ∈ {0,...,min(s_j, p-1)}.

The fiber contribution from this pair: s_j^{(1)} f_j + s_j^{(2)} f'_j = s_j f'_j + s_j^{(1)}(f_j - f'_j).

Since f_j ≠ f'_j, the set of achievable contributions is {s_j f'_j + t·(f_j - f'_j) : t = 0,...,s_j} (for s_j ≤ p-1), which has size s_j + 1.

By iterated Cauchy-Davenport over the K doubled tails (each contributing a set of size ≥ s_j + 1 to the fiber sum):

|Achievable fiber sums from doubled tails| ≥ min(p, Σ (s_j + 1) - (K-1)) = min(p, W_d + 1).

The singleton tails contribute a FIXED fiber sum F_s. The total achievable fiber sum is F_s + (achievable doubled sum). For a zero-sum lift: we need 0 ∈ F_s + {achievable set}, i.e., -F_s in the achievable set. This holds if the achievable set = F_p (size p).

**Condition:** W_d ≥ p - 1 ⟹ the achievable set covers all of F_p ⟹ lift always succeeds. **QED.**

**Corollary.** If a tail zero-sum has weight W ≤ p and uses ≤ 1 singleton tail (W_s ≤ 1), then W_d ≥ W - 1 ≥ p - 1 when W = p, giving full fiber coverage. Hence: **any weight-p tail zero-sum using at most 1 singleton tail can always be lifted.**

## THE SHARPEST BOTTLENECK

Combining Theorems 4 and 5, the Grail reduces to:

**Conjecture L (Liftability):** For any α-dispersed set of 2α+2 vectors in F_p^{r+1} and any linear form w, there exists a tail zero-sum of weight ≤ p that can be lifted to F_p^{r+1} (i.e., the fiber constraint is satisfiable).

By Theorem 5, this is GUARANTEED if:
- There exists a weight-p tail zero-sum using at most 1 singleton tail, OR
- There exists a tail zero-sum (any weight) where the total weight on doubled tails is ≥ p-1.

The obstruction: the η bound gives tail zero-sums, but to use only doubled tails we need those tails' copies pool to reach η. The count:
- By No-Tripled-Tails: each tail has ≤ 2 vectors in S
- With T total tails and K doubled: 2K + (T-K) = 2α+2, so K = 2α+2-T
- Doubled tails' copies: K(p-1) = (2α+2-T)(p-1)
- Need K(p-1) ≥ η = α(p-1)+1 for a guaranteed doubled-only zero-sum
- This requires K ≥ α+1, i.e., T ≤ α+1 — **exactly U1'!**

So: **U1' (at most α+1 tails) is equivalent to the doubled-tails-only zero-sum condition.**

**If T ≥ α+2:** the doubled tail count K ≤ α, giving K(p-1) ≤ α(p-1) < η. Not enough for a guaranteed doubled-only zero-sum. We need singletons to participate, and the split flexibility may not cover F_p.

## PARTIAL CLOSURE: THE "TWO SINGLETONS" REGIME

**Theorem 6 (Near-Optimal Dispersal).** If T = α+2 (the minimum violation of U1'), then exactly 2 singletons participate in any tail zero-sum that uses all α doubled tails, and the fiber constraint is satisfiable for all but at most 1 value of F_p.

*Proof sketch.* With T = α+2: K = α doubled, 2 singletons. The doubled pool has α(p-1) = η - 1 copies. Adding 2 singletons: α(p-1) + 2(p-1) = (α+2)(p-1) > η. The tail zero-sum uses ≤ 2 singletons. By Theorem 5: |achievable fiber sums| ≥ min(p, W_d + 1). With W_s = 2 (two singletons each contributing ≥ 1): W_d ≤ p-2, giving ≥ p-1 achievable values. This covers all of F_p except possibly 1 element.

For the fiber constraint to FAIL: the missing element must be exactly -F_s. This is a codimension-1 condition on the fiber values — it holds only for a measure-zero set of configurations.

**But we need deterministic closure.** The single missing value could be the one we need. To rule this out requires either:
1. Showing multiple distinct tail zero-sums exist (giving different F_s values), or
2. Using structural properties of α-dispersed sets to constrain fiber values.
