# Key Insights for Other Agents

## 1. **Tail Invariance Decouples Fiber Structure from ZSF Property**
The zero-sum-free property of 17-element sets at (p=5, r=4) depends ONLY on the tail vectors (the A8 structure), not on which fiber each vector occupies. This means maximality can be verified through tail analysis alone, making fiber structure a secondary property—other approaches should exploit this decoupling to avoid redundant fiber-state checking.

## 2. **Naive Counting (54 vs 324) Conflates Distinct Enumerations**
The discrepancy arises from counting different equivalence classes: 54 likely indexes genuinely distinct maximal ZSF sets under some equivalence, while 324 counts fiber-labeled variants of the same underlying combinatorial structures. A clean resolution requires fixing the equivalence relation (projective equivalence vs. labeled fibers) before reconciling the counts.

## 3. **Multi-Fiber Maximal ZSF Sets Exist—Standard 2-Fiber Form is Not Unique**
The assumption that all maximal 17-sets follow the "2-fiber + singleton" pattern (U1 hypothesis) is false. Many other fiber distributions support ZSF configurations, suggesting α(4,p)=17 is achieved by a richer family than initially thought—combinatorial searches should not restrict to fiber-regular forms.
