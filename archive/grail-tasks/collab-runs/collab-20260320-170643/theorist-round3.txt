## Theorem 1: Window Structure for A₈ on L (PROVED)

**Theorem.** Let A₈ = {a₀,...,a₇} ⊂ F_p³ with the coordinate representation yielding constraints:

```
c₀ - c₃ + c₄ + c₅ - c₆ - c₇ = 0     (C1: first coord = 0)
c₁ - c₂ + c₄ - c₅ + c₆ - c₇ = 0     (C2: 2nd = 3rd coord)
j = c₁ + c₃ + c₄ + 2c₆ + c₇          (value on L)
Σ cᵢ = w                                (weight)
cᵢ ≥ 0                                  (non-negativity)
```

Then for any weight w ∈ {2,...,p-1}: (0,j,j) ∈ Σ_w(A₈) if and only if ⌈w/2⌉ ≤ j ≤ 2⌊w/2⌋.

**Proof.** Solve (C1) and (C2) for c₀, c₁ in terms of the 6 free variables c₂,...,c₇:

```
c₀ = c₃ - c₄ - c₅ + c₆ + c₇
c₁ = c₂ - c₄ + c₅ - c₆ + c₇
```

Substituting into the value and weight expressions:

```
j = c₂ + c₃ + c₅ + c₆ + 2c₇         ...(V)
w = 2c₂ + 2c₃ - c₄ + c₅ + c₆ + 3c₇  ...(W)
```

**Step 1 — Key identity.** From (V) and (W):

```
2j − w = c₄ + c₅ + c₆ + c₇ ≥ 0
```

Hence **j ≥ w/2**, giving j ≥ ⌈w/2⌉ (since j is an integer).

**Step 2 — No wrap-around.** Since all cᵢ ≥ 0 and Σcᵢ = w, each cᵢ ≤ w ≤ p−1. The integer value of j satisfies 0 ≤ j ≤ w ≤ p−1 (from 2j−w ≤ w, since c₄+c₅+c₆+c₇ ≤ Σcᵢ = w). **No modular reduction is needed for any p ≥ w+1.**

**Step 3 — Upper bound and parity constraint.** Suppose j = w (maximal). Then:
- w − j = c₂ + c₃ + c₇ − c₄ = 0, so c₄ = c₂ + c₃ + c₇.
- 2j − w = c₄ + c₅ + c₆ + c₇ = w.
- Combining: c₀ = c₃ − (c₂+c₃+c₇) − c₅ + c₆ + c₇ = −c₂ − c₅ + c₆.
- c₁ = c₂ − (c₂+c₃+c₇) + c₅ − c₆ + c₇ = −c₃ + c₅ − c₆.

For c₀ ≥ 0: c₆ ≥ c₂ + c₅. For c₁ ≥ 0: c₅ ≥ c₃ + c₆. Combining: c₅ ≥ c₃ + c₂ + c₅, so **c₂ = c₃ = 0**. Then c₆ ≥ c₅ and c₅ ≥ c₆, so **c₅ = c₆**. Also c₄ = c₇, and c₇ = 0 (from c₄ = c₂+c₃+c₇ = c₇ and Σcᵢ = w = 2c₅+2c₇... actually c₄+c₇ = 2c₇ and c₅+c₆ = 2c₅, total = 2c₅+2c₇ = w). So **j = 2c₅ + 2c₇ is even**.

Therefore: **if w is odd, j = w is impossible**, giving j ≤ w−1 = 2⌊w/2⌋. Combined: **j ≤ 2⌊w/2⌋** for all w.

**Step 4 — Completeness (every j in the window is achieved).**

*Even w = 2k, j = k+t for t ∈ {0,...,k}*: Set c₂ = k−t, c₃ = c₄ = c₇ = 0, c₅ = c₆ = t.
- c₀ = 0−0−t+t+0 = 0 ≥ 0 ✓
- c₁ = (k−t)−0+t−t+0 = k−t ≥ 0 ✓
- j = (k−t)+0+t+t+0 = k+t ✓
- w = 2(k−t)+0−0+t+t+0 = 2k ✓

*Odd w = 2k+1, j = k+t for t ∈ {1,...,k}*: Set c₂ = k+1−t, c₃ = c₄ = c₇ = 0, c₅ = t−1, c₆ = t.
- c₀ = 0−0−(t−1)+t+0 = 1 ≥ 0 ✓
- c₁ = (k+1−t)−0+(t−1)−t+0 = k−t ≥ 0 ✓ (since t ≤ k)
- j = (k+1−t)+0+(t−1)+t+0 = k+t ✓
- w = 2(k+1−t)+0−0+(t−1)+t+0 = 2k+1 ✓ **∎**

**Corollary.** The window has width ⌊w/2⌋+1 (even w) or ⌊w/2⌋ (odd w). At w=1: empty window (no A₈ vector lies on L). At w=0: only the origin.
