# WINDOW STRUCTURE THEOREM — COMPLETE PROOF

## Setup

**A₈** = {a₀,...,a₇} ⊂ F_p³ where:
- a₀=(1,0,0), a₁=(0,1,0), a₂=(0,0,1), a₃=(-1,1,1)  [total weight φ = 1]
- a₄=(1,1,0), a₅=(1,0,1), a₆=(-1,2,1), a₇=(-1,1,2)  [total weight φ = 2]

Here φ(x,y,z) = x+y+z is the **total weight grading**.

**Invariant line:** L = {(0,j,j) : j ∈ F_p}.

**Claim:** For p ≥ 2w+1: (0,j,j) ∈ Σ_w(A₈) iff ⌈w/2⌉ ≤ j ≤ 2⌊w/2⌋.

## Proof

### Step 1: The Total Weight Identity

For any (0,j,j) = Σ cᵢaᵢ with Σcᵢ = w, apply φ:

> **2j** = φ(0,j,j) = Σ cᵢ φ(aᵢ) = (c₀+c₁+c₂+c₃)·1 + (c₄+c₅+c₆+c₇)·2 = **w + S**

where **S = c₄+c₅+c₆+c₇**.

(For p ≥ 2w+1: all cᵢ ≤ w < p, and linear combinations have absolute value < p, so this holds over **Z**, not just F_p.)

### Step 2: Parity Constraint

The two linear constraints for targeting L are:
- (I): c₀ - c₃ + c₄ + c₅ - c₆ - c₇ = 0
- (II): c₁ - c₂ + c₄ - c₅ + c₆ - c₇ = 0

**Add (I) and (II):**
> c₀ + c₁ - c₂ - c₃ + 2c₄ - 2c₇ = 0

So **c₀+c₁ - c₂-c₃ = 2(c₇-c₄)**, which is even.

Since c₀+c₁+c₂+c₃ = (c₀+c₁-c₂-c₃) + 2(c₂+c₃), and both terms are even:

> **c₀+c₁+c₂+c₃ is even**, hence **S = w - (c₀+c₁+c₂+c₃) ≡ w (mod 2)**.

### Step 3: Necessary Conditions (the window bounds)

From 2j = w + S with S ≡ w (mod 2):

**Lower bound:** S ≥ 0 ⟹ 2j ≥ w ⟹ j ≥ ⌈w/2⌉.
- (For odd w: S ≥ 1 since S is odd, giving 2j ≥ w+1, j ≥ (w+1)/2 = ⌈w/2⌉.)

**Upper bound:** S ≤ w gives j ≤ w. But can S = w when w is odd?
- S = w requires c₀ = c₁ = c₂ = c₃ = 0.
- Then (I): c₄+c₅ = c₆+c₇, and (II): c₄+c₆ = c₅+c₇.
- Subtracting: 2(c₅-c₆) = 0, so c₅ = c₆ (over **Z**). Similarly c₄ = c₇.
- Then S = 2c₄ + 2c₅ is **even**, contradicting S = w odd.
- So for odd w: S ≤ w-2, giving j ≤ (w + w-2)/2 = w-1 = 2⌊w/2⌋.

For even w: S = w is achievable (see Step 4), giving j ≤ w = 2⌊w/2⌋.

**Result:** j ∈ {⌈w/2⌉, ..., 2⌊w/2⌋} is necessary.

### Step 4: Sufficiency (explicit constructions)

**Even weight w = 2k, target j = k+t** for t ∈ {0,...,k}:

| Variable | c₀ | c₁ | c₂ | c₃ | c₄ | c₅ | c₆ | c₇ |
|----------|-----|-----|-----|-----|-----|-----|-----|-----|
| Value    | k-t | 0   | 0   | k-t | t   | 0   | 0   | t   |

- Weight: 2(k-t) + 2t = 2k ✓
- (I): (k-t) - (k-t) + t + 0 - 0 - t = 0 ✓
- (II): 0 - 0 + t - 0 + 0 - t = 0 ✓
- j = 0 + (k-t) + t + 0 + t = k+t ✓
- All cᵢ ≥ 0 ✓

**Odd weight w = 2k+1, target j = k+1+t** for t ∈ {0,...,k-1}:

| Variable | c₀ | c₁ | c₂ | c₃ | c₄ | c₅ | c₆ | c₇ |
|----------|-------|-----|-----|-----|-------|-----|-----|-----|
| Value    | k-t-1 | 0   | 1   | k-t | 1+t   | 0   | 0   | t   |

- Weight: (k-t-1) + 0 + 1 + (k-t) + (1+t) + 0 + 0 + t = 2k+1 ✓
- (I): (k-t-1) - (k-t) + (1+t) + 0 - 0 - t = 0 ✓
- (II): 0 - 1 + (1+t) - 0 + 0 - t = 0 ✓
- j = 0 + (k-t) + (1+t) + 0 + t = k+1+t ✓
- All cᵢ ≥ 0 (requires k ≥ t+1, i.e., t ≤ k-1) ✓

**QED.** ∎

### Step 5: Extension to w = p-1

The linear constraints (I), (II) have LHS in [-(p-1), p-1] when Σcᵢ = p-1, so "≡ 0 mod p" forces exact integer equality. The parity argument applies.

Now 2j ≡ S-1 (mod p) with two integer branches:
- **k=0:** 2j = S-1. Requires S odd. But parity forces S even. **Dead.**
- **k=1:** 2j = S-1+p. Requires S even ✓. Gives j = (S+p-1)/2 ∈ {(p-1)/2,...,p-1}.

Same interpolation (c₀,c₃,c₄,c₇) = ((p-1)/2-t, (p-1)/2-t, t, t) achieves j = (p-1)/2 + t.

**Corollary (Gap at weight p-1 on L):** {(0,j,j) : 1 ≤ j ≤ (p-3)/2}, size (p-3)/2.

**Corollary (Shadow saturation on L):** ∪_{w=2}^{p-1} (Σ_w ∩ L) = L \ {0}, since the windows tile {1,...,p-1} completely.
