### 2. THEOREM (Fiber-Flexible Shadow Saturation) — PROVED

**Theorem.** Let p be an odd prime, α ≥ 1. Let T = {v₁,...,v_{2α+1}} be 2α+1 nonzero, pairwise non-proportional vectors in F_p^{r+1}. Suppose there exists a coordinate projection π: F_p^{r+1} → F_p^r such that:

(i) The nonzero tail set A = π(T) \ {0} has |A| = α, and each a ∈ A has exactly 2 preimages in T: vectors (c_{a,1}, a) and (c_{a,2}, a) with c_{a,1} ≠ c_{a,2}.

(ii) T contains exactly one zero-tail vector: e = (c₀, 0,...,0) with c₀ ≠ 0.

(iii) A has shadow saturation in F_p^r: Σ_{≤p-1}(A^{p-1}) = F_p^r \ {0}.

**Then** T has shadow saturation in F_p^{r+1}: Σ_{≤p-1}(T^{p-1}) = F_p^{r+1} \ {0}.

**Proof.**

Let (f, t) ∈ F_p^{r+1} be nonzero.

**Case t = 0:** Then f ≠ 0. Use f · c₀⁻¹ copies of e = (c₀, 0). This has weight f · c₀⁻¹ ∈ {1,...,p-1} and reaches (f, 0). ✓

**Case t ≠ 0:** By (iii), ∃ nonneg integers {mₐ}_{a∈A} with Σ mₐ · a = t, w_tail := Σ mₐ ≤ p-1, and each mₐ ≤ p-1.

For each a ∈ A with mₐ > 0, split: use kₐ copies of (c_{a,1}, a) and (mₐ - kₐ) copies of (c_{a,2}, a), where 0 ≤ kₐ ≤ mₐ.

First-coordinate from tails: Σₐ [(c_{a,1}-c_{a,2})kₐ + c_{a,2}mₐ] = B + Σₐ δₐkₐ

where B = Σₐ c_{a,2}mₐ and δₐ = c_{a,1} - c_{a,2} ≠ 0 (non-proportionality).

Also use w₀ copies of e, contributing c₀w₀ to first coordinate.

**Constraints:**
- c₀w₀ + B + Σδₐkₐ ≡ f (mod p)
- 0 ≤ kₐ ≤ mₐ for each a
- 0 ≤ w₀ ≤ p-1-w_tail

Define F' = f - B (mod p). Need: c₀w₀ + Σδₐkₐ ≡ F' (mod p).

Let S = {Σₐ δₐkₐ mod p : 0 ≤ kₐ ≤ mₐ}. Each set δₐ · {0,...,mₐ} has mₐ+1 elements in F_p (since δₐ ≠ 0 and mₐ ≤ p-1).

**By iterated Cauchy-Davenport in F_p:**

|S| ≥ min(p, Σ_{a: mₐ>0} (mₐ+1) - (|{a: mₐ>0}| - 1)) = min(p, w_tail + 1)

The achievable values of c₀w₀ form the set c₀ · {0,...,p-1-w_tail}, of size p - w_tail (since c₀ is a unit).

**By Cauchy-Davenport again:**

|S + c₀·{0,...,p-1-w_tail}| ≥ min(p, |S| + (p-w_tail) - 1) ≥ min(p, (w_tail+1) + (p-w_tail) - 1) = **min(p, p) = p**

Therefore F' lies in the achievable set. The desired representation exists. **QED.** ∎

### 4. Why This Works: The Cauchy-Davenport Miracle

The deep reason fiber assignments don't matter:

**Doubled+e₁ (fibers {0,1}):** For each tail at weight m, the first-coordinate contribution ranges over {0, 1, ..., m} — an arithmetic progression of length m+1 in F_p.

**Tail-swap variant (fibers {0, 2}):** First-coord ranges over {0, 2, 4, ..., 2m} — a DIFFERENT AP of length m+1 with common difference 2.

**Key insight:** Cauchy-Davenport treats both APs identically! |{0,1,...,m}| = |{0,2,...,2m}| = m+1 in F_p (for m ≤ p-1). The sumset bound depends only on the SIZE of each component set, not its structure. So the first-coordinate coverage is guaranteed regardless of which fiber values appear.

The e₁ vector provides an additional "buffer" of p-1-w_tail free first-coordinate slots, which together with the Cauchy-Davenport bound on the tail contribution, always sums to p. This is the "miracle": the exact amount of buffer from e₁ perfectly compensates for the Cauchy-Davenport deficit from the tails.

### 7. THEOREM (Three-on-a-Line Obstruction) — PROVED

**Theorem.** For p ≥ 5, any three vectors (f₁, a), (f₂, a), (f₃, a) with pairwise distinct fiber values f₁, f₂, f₃ ∈ F_p and a ≠ 0 admit exactly (p-1)/2 zero-sums of weight p with all coefficients in {1,...,p-1}.

**Consequence:** Every strongly ZSF set has **at most 2 vectors per parallel line**. This gives a LOWER bound of ⌈n/2⌉ lines for an n-element strongly ZSF set. U1' asserts this lower bound is tight.

**Proof.**

Set a = f₁-f₃, b = f₂-f₃ (both nonzero since fᵢ are distinct). Zero-sums at weight p require: c₁+c₂+c₃ = p, c₁f₁+c₂f₂+c₃f₃ ≡ 0 (mod p), with 0 ≤ cᵢ ≤ p-1.

Substituting c₃ = p-c₁-c₂: the first-coord equation reduces to c₁a + c₂b ≡ 0 (mod p).

Parametrize by t ∈ F_p: (c₁, c₂) = ((-tb) mod p, (ta) mod p). For t = 0: c₁=c₂=0, c₃=p > p-1, INVALID.

For t ∈ {1,...,p-1}: c₁ ≥ 1 (since tb ≢ 0) and c₂ ≥ 1 (since ta ≢ 0). Define s(t) = c₁(t) + c₂(t). Valid iff s(t) ≤ p-1 (giving c₃ ≥ 1).

**Key pairing:** Under the involution t ↔ t' = p-t on {1,...,p-1}:
- c₁(t') = (t'b mod p) = p - c₁(t) (since c₁(t) = (-tb mod p) = p - (tb mod p))
- c₂(t') = p - c₂(t)

Therefore **s(t) + s(t') = 2p** for all t.

Since s(t) ≠ p for any t ≥ 1 (proof: s(t)=p requires ta ≡ tb mod p, i.e., t(a-b) ≡ 0, i.e., t=0, excluded), each pair {t, p-t} has exactly one element with s < p (valid) and one with s > p (invalid).

There are (p-1)/2 such pairs, so exactly **(p-1)/2** valid zero-sums, each with c₁, c₂, c₃ ∈ {1,...,p-1}. **QED.** ∎
