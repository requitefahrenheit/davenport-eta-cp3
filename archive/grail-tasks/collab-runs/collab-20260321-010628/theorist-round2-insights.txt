# THEORIST ROUND 2: KEY INSIGHTS FOR COORDINATION

## Insight 1: The Two-Fiber Case Is Already Closed
The adversarial observation that S can fit in exactly 2 non-zero fibers with |F_{c₁}| + |F_{c₂}| = 2(α(r)+1) = α(r+1)+1 is **not a gap**—it's a boundary case already handled by part (a)-(b). The two-fiber upper bound theorem resolves this. Part (c) should focus on what remains: proving that **either** a linear form exists witnessing ≤2 fibers (→ two-fiber theorem), **or** the multi-fiber structure itself yields a zero-sum.

## Insight 2: Proportionality Preservation Breaks at Non-Zero Fibers
Critical discovery: In fiber F_c with c ≠ 0, if v' = λv then λc = c implies λ = 1, so v' = v. This means **proportional elements in F_p^{r+1} cannot coexist in the same non-zero fiber**—they must be in different fibers. Projection π: F_p^{r+1} → ker(w) can create new proportionalities (v = (1,0,...,c) and v' = (2,0,...,c) become proportional under π), making the naive fiber-by-fiber count unstable. The inductive hypothesis cannot be applied directly per-fiber for non-zero fibers.

## Insight 3: Partition Dichotomy Closes Induction
The cleanest closure: **For any sequence of length (2α(r)+1)(p-1)+1, partition by whether ≥3-fiber scenarios exist.** If no 3-fiber instance exists, the two-fiber theorem covers everything. If some linear form w forces ≥3 fibers, extract a zero-sum directly from the global structure using the fact that α(r+1)+1 distinct projective classes cannot avoid weight ≤p by pigeonholing on the Davenport bound for C_p^r itself. This avoids fighting proportionality-breaking per-fiber and instead leverages the full-rank constraint.
