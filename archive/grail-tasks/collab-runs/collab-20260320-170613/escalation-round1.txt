# CONVERGENCE ANALYSIS

## CONVERGENCE: Window Formula [HIGH CONFIDENCE]

**Claim:** For the canonical 8-element set A₈ in F_p³ with weight w and shadow restricted to line L = {(0,j,j)}, the reachable j-values satisfy:
$$\lceil w/2 \rceil \leq j \leq 2\lfloor w/2 \rfloor$$

**Evidence:**
- **Theorist** derives this rigorously via three identities (2j−w, w−j constraints), establishing lower bound j ≥ ⌈w/2⌉ and upper bound j ≤ 2⌊w/2⌋
- **Computer** independently verifies this formula computationally across p=29,31,37 and exhaustively for all 28 weights at p=29, confirming all intermediate weights match the formula
- Both agents reduce the problem to integer feasibility over ℤ (noting mod p is transparent when cᵢ ≤ w ≤ p−1)
