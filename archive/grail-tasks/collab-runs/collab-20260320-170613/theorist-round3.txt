# THEORIST ROUND 3: NEW RESULTS

## Result 1: WINDOW THEOREM (PROVED)

**Theorem (Shadow Window Structure).** Let A₈ = {a₀,...,a₇} ⊂ F_p³ be the standard 8-vector ZSF set, and L = {(0,j,j) : j ∈ F_p*}. For any weight w with 2 ≤ w ≤ p-1 and p ≥ 2w+1:

> Σ_w(A₈) ∩ L = {(0,j,j) : ⌈w/2⌉ ≤ j ≤ 2⌊w/2⌋}

**Proof.** The vectors of A₈ are:
- a₀=(1,0,0), a₁=(0,1,0), a₂=(0,0,1), a₃=(-1,1,1)
- a₄=(1,1,0), a₅=(1,0,1), a₆=(-1,2,1), a₇=(-1,1,2)

A point (0,j,j) ∈ Σ_w(A₈) iff there exist c₀,...,c₇ ≥ 0 with Σcᵢ = w satisfying:
```
(1)  c₀ - c₃ + c₄ + c₅ - c₆ - c₇ = 0    [first coord = 0]
(2)  c₁ - c₂ + c₄ - c₅ + c₆ - c₇ = 0    [second = third coord]
(3)  j = c₁ + c₃ + c₄ + 2c₆ + c₇          [value on L]
```

For p ≥ 2w+1, each cᵢ ≤ w ≤ (p-1)/2 < p-1, so the modular constraint cᵢ ≤ p-1 is automatic and j < p (no wraparound). This is pure integer LP.

**Elimination.** Use (1),(2) to express c₂, c₃ and the weight equation to express c₇ in terms of 5 free variables c₀, c₁, c₄, c₅, c₆ ≥ 0:
```
c₂ = w - 2c₀ - c₁ - 2c₄ - 2c₅         ... (B)
c₃ = w - c₀ - 2c₁ - 2c₄ - 2c₆         ... (C)
c₇ = 2c₀ + 2c₁ + 3c₄ + c₅ + c₆ - w    ... (A)
j  = c₀ + c₁ + 2c₄ + c₅ + c₆
```

The feasibility constraints are (A) ≥ 0, (B) ≥ 0, (C) ≥ 0, and c₀,c₁,c₄,c₅,c₆ ≥ 0.

### Lower bound: j ≥ ⌈w/2⌉

From j = c₀+c₁+2c₄+c₅+c₆ and non-negativity of c₅,c₆:
> j ≥ c₀ + c₁ + 2c₄  ... (*)

From (A) ≥ 0 rearranged: c₀+c₁+c₄ ≥ w - (c₀+c₁+2c₄+c₅+c₆) = w - j, so:
> j ≥ w - (c₀+c₁+c₄)  ... (**)

Adding (*) and (**): **2j ≥ w + c₄ ≥ w**, giving **j ≥ ⌈w/2⌉**. □

### Upper bound: j ≤ 2⌊w/2⌋

From (B) and (C): c₅ ≤ (w-2c₀-c₁-2c₄)/2 and c₆ ≤ (w-c₀-2c₁-2c₄)/2.

So j = c₀+c₁+2c₄+c₅+c₆ ≤ c₀+c₁+2c₄ + w - (3c₀+3c₁+4c₄)/2 = **w - (c₀+c₁)/2**.

For even w: j ≤ w. For j = w, need c₀ = c₁ = 0 and c₅ = c₆ = (w-2c₄)/2 must be integer. With even w this works. ✓

For odd w: suppose j = w. Then c₀ = c₁ = 0 and c₅ = c₆ = (w-2c₄)/2. But w is odd, so w-2c₄ is odd, making (w-2c₄)/2 non-integer. **Contradiction.** So j ≤ w-1 = 2⌊w/2⌋. □

### All intermediate values achieved

**Explicit construction for every j ∈ [⌈w/2⌉, 2⌊w/2⌋]:**

**Even w = 2k**, j = k + q, q ∈ {0,...,k}: Set
```
c₀ = c₃ = k-q,  c₅ = c₆ = q,  c₁ = c₂ = c₄ = c₇ = 0
```
Verification: weight = 2(k-q) + 2q = 2k ✓, constraints (1),(2) = 0 ✓, j = (k-q) + 2q = k+q ✓, all cᵢ ≥ 0 for q ≤ k ✓.

**Odd w = 2k+1**, j = k+1+q, q ∈ {0,...,k-1}: Set
```
c₀ = 1,  c₁ = k-q,  c₂ = k-1-q,  c₃ = 0,
c₄ = 0,  c₅ = q,  c₆ = q,  c₇ = 1
```
Verification: weight = 1+(k-q)+(k-1-q)+0+0+q+q+1 = 2k+1 ✓, constraint (1): 1-0+0+q-q-1 = 0 ✓, constraint (2): (k-q)-(k-1-q)+0-q+q-1 = 0 ✓, j = (k-q)+0+0+2q+1 = k+q+1 ✓, all cᵢ ≥ 0 for q ≤ k-1 ✓. **□**

## Result 2: UNIVERSAL SUMSET IDENTITY (PROVED — previously treated as computational)

**Theorem.** For ANY set A ⊆ F_p^n and any j, k ≥ 0 with j+k ≤ p-1:

> Σ_j(A) + Σ_k(A) = Σ_{j+k}(A)

**Proof.**

(⊇) Take v ∈ Σ_{j+k}(A): v = Σ cᵢaᵢ with Σcᵢ = j+k, 0 ≤ cᵢ ≤ p-1. Split (cᵢ) greedily: assign c'ᵢ = min(cᵢ, remaining quota for j), c''ᵢ = cᵢ - c'ᵢ. Then Σc'ᵢ = j, Σc''ᵢ = k, and v ∈ Σ_j + Σ_k.

(⊆) Take u ∈ Σ_j, w ∈ Σ_k: u = Σc'ᵢaᵢ, w = Σc''ᵢaᵢ with Σc'ᵢ = j, Σc''ᵢ = k. Then u+w = Σ(c'ᵢ+c''ᵢ)aᵢ with total weight j+k. **Key bound**: c'ᵢ ≤ j and c''ᵢ ≤ k (since non-negative coefficients sum to j resp. k), so c'ᵢ + c''ᵢ ≤ j+k ≤ p-1. Thus u+w ∈ Σ_{j+k}(A). **□**

**Significance:** This was listed as a computational observation ("verified p ≤ 23"). It is in fact **trivially true for all sets, all primes**. The proof requires only that coefficients are non-negative integers bounded by p-1, which holds whenever j+k ≤ p-1.

## Result 3: U1 REDUCTION THEOREM (NEW)

**Theorem (Cross-Fiber Zero-Sum Reduction).** For the (8,8,1) distribution with A₈ in fibers 0 and 1 and vector b = (c, v) in fiber c, proving U1 reduces to:

> For every c ∈ {2,...,p-1} and v ∈ F_p³ \ {0}, there exists d ∈ {1,...,p-1} such that -dv ∈ Σ_s(A₈) for some s with p - (dc mod p) ≤ s ≤ p - d.

**Proof of reduction.** A zero-sum of weight w = w₀ + w₁ + d ≤ p uses coefficient d on the fiber-c vector, w₁ from fiber 1, w₀ from fiber 0. First coordinate: w₁ + dc ≡ 0 (mod p), forcing w₁ = p - (dc mod p). Last three coordinates: -dv ∈ Σ_{w₀}(A₈) + Σ_{w₁}(A₈) = Σ_{w₀+w₁}(A₈) (by Result 2). Weight constraint w₀+w₁+d ≤ p gives s = w₀+w₁ ∈ [p-(dc mod p), p-d]. □

**Key observation:** As d ranges over {1,...,p-1}, the target -dv traces out all p-1 nonzero scalar multiples of v. By shadow saturation, each -dv ∈ Σ_{s(d)}(A₈) for some s(d) ∈ {2,...,p-1}. The question is whether, for at least one d, s(d) falls in the "allowed window" [p - (dc mod p), p - d].

- Gap of Σ_{p-1}: {(0,j,j) : 1 ≤ j ≤ (p-3)/2}
- Gap of -Σ_{p-1}: {(0,j,j) : (p+3)/2 ≤ j ≤ p-1}

Their union is {1,...,(p-3)/2} ∪ {(p+3)/2,...,p-1}, and the covered region is:
> Σ_{p-1}(A₈) ∪ (-Σ_{p-1}(A₈)) ⊇ L

**Proof.** The Window Theorem gives Σ_{p-1} ∩ L = {j : (p-1)/2 ≤ j ≤ p-1}. Negation maps j to p-j (mod p), so -Σ_{p-1} ∩ L = {j : 1 ≤ j ≤ (p+1)/2}. Their union is {1,...,p-1} = all of L. □

**Consequence for U1:** For the coefficient d and its "complement" p-d, the targets -dv and -(p-d)v = dv cover complementary shadow regions on L. This suggests that for MOST (c,v) pairs, either d or p-d provides a valid zero-sum.

## Result 6: WEIGHT-COMPATIBILITY THEOREM (NEW — KEY TO U1)

**Theorem (Weight-Compatibility for c = 2 on L).** For any prime p ≥ 5, any j ∈ {1,...,(p-3)/2}, and c = 2: there exists d ∈ {1,...,(p-1)/2} such that (0, dj, dj) ∈ Σ_s(A₈) for some s with p - 2d ≤ s ≤ p - d.

**Proof.**

**Step 1: Window containment analysis.** By the Window Theorem (which holds for ALL 2 ≤ w ≤ p-1 since coefficient bounds cᵢ ≤ p-1 are automatic when Σcᵢ = w ≤ p-1), the value m ∈ {1,...,p-1} satisfies (0,m,m) ∈ Σ_s(A₈) iff ⌈s/2⌉ ≤ m ≤ 2⌊s/2⌋.

The set of weights s compatible with target m is:
```
S(m) = {s : ⌈s/2⌉ ≤ m ≤ 2⌊s/2⌋} ⊇ [m, 2m] ∩ Z
```

**Step 2: Interval intersection.** For c = 2, d ∈ {1,...,(p-1)/2}, the allowed weight interval is I(d) = [p-2d, p-d]. Target is m = dj (no mod-p wraparound since dj ≤ ((p-1)/2)·((p-3)/2) < p² and we'll ensure dj < p).

The intersection S(dj) ∩ I(d) is nonempty iff:
- **dj ≤ p - d** (target below right endpoint of I), i.e., **d(j+1) ≤ p**
- **2dj ≥ p - 2d** (twice target above left endpoint of I), i.e., **d(2j+2) ≥ p**

Combined: **p/(2j+2) ≤ d ≤ p/(j+1)**.

**Step 3: Integer existence in the interval.** The interval [p/(2j+2), p/(j+1)] has length:
```
p/(j+1) - p/(2j+2) = p/(2j+2)
```

For j ≤ (p-3)/2: p/(2j+2) ≥ p/(p-1) > 1. So the interval has length > 1 and **always contains at least one integer** d.

**Step 4: Range verification.** For d = ⌈p/(2j+2)⌉:
- d ≤ p/(2j+2) + 1. Since j ≥ 1: d ≤ p/4 + 1 ≤ (p-1)/2 for p ≥ 5. ✓
- d(j+1) ≤ (p/(2j+2) + 1)(j+1) = p/2 + j + 1 < p (since j ≤ (p-3)/2). ✓
- dj < p (no modular wraparound). ✓ **□**

## ADDENDUM: Connection to Literature

The Sumset Identity (Result 2) can be seen as a trivial case of Grynkiewicz's weighted EGZ framework: when coefficients are unrestricted in {0,...,p-1} and the total weight j+k ≤ p-1, the splitting property is automatic. The deeper weighted EGZ theorem handles the case where weights are prescribed (not free), which is harder.
