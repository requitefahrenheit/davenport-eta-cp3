## THEOREM A (Inductive Fiber Structure)

**Statement.** Assume Q(r-1) for r ≥ 4, p ≥ 5. Let B be a ZSF set of size 2α+2 in F_p^r. Then for any linear functional π: F_p^r → F_p:

(i) Every fiber F_c = {v ∈ B : π(v) = c} satisfies |F_c| ≤ α.

(ii) B occupies at least 3 fibers.

(iii) For any two non-proportional vectors v, w ∈ B, there exists π with π(v) = 1, π(w) = -1 (complementary fiber placement).

**Proof.**

*(i) Fiber 0:* F_0 ⊂ ker(π) ≅ F_p^{r-1}. Vectors in F_0 are pairwise non-proportional (inherited from B). If |F_0| ≥ α+1, the sequence (F_0, mult p-1) has length (α+1)(p-1) ≥ α(p-1)+1 = η(C_p^{r-1}), yielding a weight-≤-p zero-sum that lifts directly to F_p^r, contradicting ZSF(B).

*(i) Fiber c ≠ 0:* Vectors v_i = (c, t_i) in F_c have pairwise non-proportional tails t_i (since (c,t) = λ(c,t') forces λ=1, t=t'). If |F_c| ≥ α+1, the tails form a sequence of length ≥ η(C_p^{r-1}), giving a tail zero-sum Σa_i t_i = 0 at weight w ≤ p. Lifting: Σa_i(c,t_i) = (cw, 0). Since c ≠ 0, this vanishes iff w ≡ 0 (mod p), forcing w = p. This is a weight-p zero-sum in F_p^r, contradicting ZSF(B).

*(ii)* Two fibers give total ≤ 2α < 2α+2. Contradiction.

*(iii)* Since v, w are non-proportional and nonzero, they are linearly independent. Extend to a basis and define π(v) = 1, π(w) = -1, π(e_i) = 0 for remaining basis vectors. □

## THEOREM B (Shadow Propagation)

**Statement.** For all r ≥ 3, p ≥ 5: if η(C_p^r) = α(r)(p-1)+1, then every ZSF set A of size α(r) in F_p^r satisfies Σ_{≤p-1}(A) = F_p^r \ {0}.

That is: proving η at rank r automatically gives shadow saturation at rank r.

**Proof.** For nonzero t ∈ F_p^r:

*Case 1: -t is proportional to some v ∈ A.* Say -t = λv. Then t = (p-λ)v with weight p-λ ≤ p-1. So t ∈ Σ_{p-λ}(A).

*Case 2: -t is not proportional to any v ∈ A.* Form the sequence S = A^{p-1} · (-t)^1 (each vector of A with multiplicity p-1, plus one copy of -t). Length = α(r)(p-1) + 1 = η(C_p^r). By the η definition, S has a nonempty zero-sum of weight ≤ p. Since A is ZSF, the zero-sum must involve -t. So t = Σa_i v_i for some a_i with Σa_i ≤ p-1, giving t ∈ Σ_{≤p-1}(A). □

**Corollary.** The strengthened inductive hypothesis Q(r) is self-consistent: Q(3) holds (base case, proved and Lean-verified). For r ≥ 4: Q(r-1) + [η at rank r] → Q(r). The entire Grail reduces to proving η at each rank.

## THEOREM C (Weight-p Barrier — restated cleanly)

**Statement.** Let B be ZSF in F_p^r. Fix π. Any zero-sum using vectors exclusively from a single fiber c ≠ 0 has integer weight ≡ 0 (mod p), hence weight ≥ p.

**Proof.** Vectors v_i = (c, t_i). Zero-sum Σa_i v_i = 0 requires c · (Σa_i) = 0 in F_p. Since c ≠ 0: Σa_i ≡ 0 (mod p). As integers, weight ≥ p. □

**Corollary.** Zero-sums of weight < p MUST be cross-fiber (involve vectors from ≥ 2 fibers with distinct π-values).

## THEOREM D (The Tail Decomposition)

**Statement.** Fix π: F_p^r → F_p and a complement vector u with π(u) = 1. Write v = π(v)·u + t(v) where t(v) ∈ ker(π). For vectors v ∈ F_c and w ∈ F_{-c} (complementary fibers):

t(v) + t(w) = v + w

and v + w ∈ ker(π).

**Proof.** t(v) = v - c·u and t(w) = w - (-c)·u = w + c·u. Sum: t(v) + t(w) = v + w. And π(v+w) = c + (-c) = 0, so v + w ∈ ker(π). □

## THEOREM E (The Cross-Fiber Pigeonhole — MAIN NEW RESULT)

**Statement.** Assume Q(r-1). Let B be ZSF of size 2α+2 in F_p^r. Choose π placing v ∈ F_1, w ∈ F_{p-1} (complementary fibers, exists by Theorem A(iii)). Let F_0 = B ∩ ker(π).

If |F_0| = α (maximal) and

**|T_1| + |T_{p-1}| > |M(F_0)| + 1**

where T_c = tails of fiber c and M(F_0) is the missing layer of F_0, then B contains a zero-sum of weight ≤ p, contradicting ZSF.

**Proof.** By Theorem D, for each v' ∈ F_1, w' ∈ F_{p-1}: v' + w' ∈ ker(π) \ {0} (nonzero since v', w' are non-proportional: if v' + w' = 0 then w' = -v', so (p-1, t_{w'}) = -(1, t_{v'}) = (p-1, -t_{v'}), giving t_{w'} = -t_{v'}, meaning w' = -v' — they're proportional, contradicting B non-proportional).

By Q(r-1) (shadow saturation): Σ_{≤p-1}(F_0) = F_p^{r-1} \ {0}. So for each such pair, -(v'+w') ∈ Σ_{≤p-1}(F_0).

If -(v'+w') ∈ Σ_{≤p-2}(F_0): the fiber-0 representation has weight ≤ p-2. Combined with v' + w' (weight 2), total weight ≤ p. Done.

If -(v'+w') ∈ M(F_0): bad — fiber-0 weight is p-1, total p+1. Off by 1.

The set of candidates: {-(v'+w') : v' ∈ F_1, w' ∈ F_{p-1}} = -(T_1 + T_{p-1}) in ker(π).

By Kneser's theorem in F_p^{r-1}: |T_1 + T_{p-1}| ≥ |T_1| + |T_{p-1}| - 1 (period is trivial since |T_1| + |T_{p-1}| ≤ 2α < p^{r-1}).

|Σ_{≤p-2}(F_0)| = (p^{r-1} - 1) - |M(F_0)|.

By pigeonhole, the intersection is nonempty when:

|Σ_{≤p-2}(F_0)| + |T_1 + T_{p-1}| > p^{r-1} - 1

⟺ (p^{r-1} - 1 - |M|) + (|T_1| + |T_{p-1}| - 1) > p^{r-1} - 1

## STRUCTURAL INSIGHT: WHY THE GAP IS EXACTLY 1

The weight-p+1 barrier is not accidental. Here is the chain:

1. **Weight-p barrier** (Theorem C): intra-fiber zero-sums have weight ≡ 0 mod p, so weight ≥ p.
2. **Complementary fibers**: minimum cross-fiber weight contribution = 2 (one vector from each of fibers c, -c).
3. **Shadow saturation** (Theorem B): targets representable from fiber 0 at weight ≤ p-1.
4. **Combined**: 2 + (p-1) = p+1. Off by exactly 1.

The missing layer M is the set of targets that "use up" the full weight budget p-1 from fiber 0. If M is small (≤ (p-3)/2), there's room to avoid it via the pigeonhole across the ≥ (p-1)/2 cross-fiber candidates. The bound |M| = (p-3)/2 is TIGHT (achieved by A8), making this a razor-thin argument.
