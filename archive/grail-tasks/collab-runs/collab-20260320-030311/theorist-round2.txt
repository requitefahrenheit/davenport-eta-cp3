## THEOREM 1: Shadow Saturation from η

**Statement.** If η(C_p^r) = α(r)(p-1)+1 and A is a ZSF set of size α(r) in F_p^r, then Σ_{p-1}(A) = F_p^r \ {0}.

**Proof.** Fix any nonzero v ∈ F_p^r. Form the sequence A^{p-1} · (-v), length α(r)(p-1)+1 = η(C_p^r). By definition of η, this has a zero-sum subsequence of length ≤ p. Since A^{p-1} is ZSF, the element -v must participate:

> Σc_i·a_i + (-v) = 0, with c_i ∈ {0,...,p-1}, Σc_i ≤ p-1.

Rearranging: **v = Σc_i·a_i** with Σc_i ≤ p-1. Since v ≠ 0, we have Σc_i ≥ 1, so v ∈ Σ_{p-1}(A).

*Subtlety*: if -v ∈ A, say -v = a_j, the η argument only gives the trivial zero-sum p·a_j. But then v = (p-1)·a_j ∈ Σ_{p-1}(A) at weight p-1 trivially. (And -v ∉ A for non-proportional reasons when v is a non-trivial scalar multiple of a_j.) **□**

## THEOREM 2: Shadow Reduction

**Statement.** If Σ_{p-1}(A) = F_p^r \ {0}, then the doubled+e1 S = {(0,a): a ∈ A} ∪ {(1,a): a ∈ A} ∪ {e₁} satisfies Σ_{p-1}(S) = F_p^{r+1} \ {0}.

**Proof.** Target (x₀, y) ≠ 0. Use coefficients f_i on a_i (split between fibers) and e on e₁.

- Weight = Σf_i + e (since fiber-0 and fiber-1 contributions to a_i combine additively).
- First coordinate: choose e = max(0, x₀ - Σf_i). Distribute d_i ≤ f_i with Σd_i = x₀ - e.
- **Minimum weight = max(Σf_i, x₀) ≤ p-1.**

**Case y ≠ 0:** By hypothesis, ∃ f with Σf_i·a_i = y, 1 ≤ Σf_i ≤ p-1. Weight = max(Σf_i, x₀) ≤ p-1. ✓

**Case y = 0, x₀ ≥ 1:** Use e₁ alone with coefficient x₀. Weight = x₀ ≤ p-1. ✓ **□**
