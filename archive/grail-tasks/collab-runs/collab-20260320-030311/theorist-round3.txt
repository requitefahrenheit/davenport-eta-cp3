has **no zero-sum of weight ≤ p**. That is, S^{p-1} has no zero-sum subsequence of length in [1, p].

**PROOF.** The copies pool A^{p-1} has length n(p-1) = η(C_p^r) − 1 < η(C_p^r). By definition of η, A^{p-1} contains **no** zero-sum subsequence of length in [1, p].

Suppose for contradiction that S^{p-1} has a zero-sum of weight w ∈ [1, p]:

∑ cᵢ·(0, aᵢ) + ∑ dᵢ·(1, aᵢ) + e·e₁ = 0,   cᵢ, dᵢ, e ∈ {0,...,p−1},   w = ∑cᵢ + ∑dᵢ + e.

**First coordinate:** ∑dᵢ + e ≡ 0 (mod p). Since ∑dᵢ + e ≤ w ≤ p and ∑dᵢ + e ≥ 0, either ∑dᵢ + e = 0 or ∑dᵢ + e = p.

**Case A: ∑dᵢ + e = 0.** All dᵢ = 0, e = 0. Tail equation: ∑cᵢ·aᵢ = 0 with ∑cᵢ = w ∈ [1, p]. This is a zero-sum of A^{p-1} of length ≤ p. **Contradicts η-subthreshold.**

- If e = 0: ∑dᵢ = p, a weight-p zero-sum of A^{p-1}. **Contradicts η-subthreshold.**
- If e ≥ 1: ∑dᵢ = p − e ≤ p − 1, a zero-sum of A^{p-1} of length ≤ p − 1. **Contradicts η-subthreshold.**

All cases lead to contradiction. ∎

## PART II: FIBER STRUCTURE CONSTRAINTS

**THEOREM B (Fiber Bound for Strongly ZSF Sets).** Let S be a set of n = 2α(r)+1 pairwise non-proportional nonzero vectors in F_p^{r+1} such that S^{p-1} has no zero-sum of length ≤ p. Let w: F_p^{r+1} → F_p be any linear form, with fibers F_c = {v ∈ S : w(v) = c}. Then:

**(i) Zero fiber:** |F₀| ≤ α(r).

**(ii) Nonzero fiber, all nonzero tails:** If every v ∈ F_c (c ≠ 0) has nonzero projection to ker(w), then |F_c| ≤ α(r).

**(iii) Nonzero fiber with zero-tail:** If F_c (c ≠ 0) contains exactly one vector with zero projection to ker(w), then |F_c| ≤ α(r)+1.

**PROOF.**

**(i):** The tails (projections to ker(w) ≅ F_p^r) of F₀ vectors are nonzero (else the vector is zero). A zero-sum of the tail copies pool at weight ≤ p lifts to a zero-sum of S at the same weight (first coordinate is automatically 0). The copies pool has |F₀|·(p−1) elements. If |F₀| ≥ α(r)+1: length ≥ (α(r)+1)(p−1) ≥ η(C_p^r). A zero-sum of length ≤ p exists. Contradiction.

**(ii):** Suppose |F_c| = m with all tails nonzero. A weight-≤-p zero-sum restricted to F_c has first coordinate c·∑dᵢ ≡ 0 (mod p), so ∑dᵢ ≡ 0 (mod p). With ∑dᵢ ≤ p: either ∑dᵢ = 0 (trivial) or ∑dᵢ = p. For the latter: ∑dᵢ·tail(vᵢ) = 0 is a weight-p zero-sum of the tails. But we also need to check weight-≤-p zero-sums involving vectors from OTHER fibers. By the same first-coordinate argument as Theorem A (Case B), any zero-sum involving F_c vectors at total weight ≤ p must use ONLY F_c vectors (see below). So the tails' copies pool of length m(p-1) must avoid length-≤-p zero-sums. If m ≥ α(r)+1: length ≥ η(C_p^r), contradiction.

**Cross-fiber impossibility at weight ≤ p:** Suppose a weight-≤-p zero-sum involves vectors from fibers F_a and F_b (a ≠ b). First coordinate: a·∑d^a_i + b·∑d^b_j + ... ≡ 0 (mod p). The integer value of this sum is in [0, p]. But different fiber contributions create a non-trivial linear equation mod p, and the integer constraint forces each "fiber group" to satisfy its own mod-p condition independently. Formally, the argument of Theorem A applies: the total first-coordinate contribution ∑(fiber-c contributions)·c must be 0 mod p with integer value ≤ p, which forces either all-zero or exactly-p. This partitions any weight-≤-p zero-sum into single-fiber contributions.

**(iii):** The zero-tail vector doesn't contribute to the tail copies pool. The nonzero tails number |F_c|−1. The argument of (ii) gives |F_c|−1 ≤ α(r), so |F_c| ≤ α(r)+1. ∎

**THEOREM B' (The s-Extremal Insight).** The fiber-1 of the doubled+e1 has α(r)+1 vectors: α(r) with tails matching A, plus one with zero tail (e₁). The tail copies pool (including zero tails as inert elements) is exactly:

> A^{p-1} ∪ {0^{p-1}} = extremal sequence for s(C_p^r).

The 0 copies are inert (don't participate in sums), so the sequence avoids length-p zero-sums iff A^{p-1} does — which is guaranteed by η.

**This is the structural reason the doubled+e1 works:** it places the critical fiber at the **exact s-extremal threshold**, using the zero-tail vector to occupy one slot without contributing to the tail copies pool.
