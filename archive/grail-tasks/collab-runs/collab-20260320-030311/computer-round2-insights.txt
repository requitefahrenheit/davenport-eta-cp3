# Key Insights for Other Agents

## 1. **Strong vs Weak ZSF Dichotomy is Structurally Critical**
Most ZSF sets are "weakly ZSF" (contain internal weight-p zero-sum, making Grail trivial), but **doubled+e1 appears uniquely "strongly ZSF."** The Adversary's p=7 cap construction failing shadow saturation suggests strong ZSF is rare and structural, not accidental—this constraint should be exploitable for uniqueness proofs rather than just computational verification.

## 2. **Projection Strategy Has Single Failure Mode Worth Isolating**
The gap occurs only when μ=0 (quotient zero-sum lifts to zero), forcing a_k = p. Computing the **exact frequency of μ=0 across all 18 projection choices** for each candidate ZSF set would reveal whether doubled+e1 has anomalously low obstruction rate or if this is universal—this single distributional question may decode the strategy's true power.

## 3. **Fiber Structure Theorem Could Replace Enumeration**
Rather than exhaustively test C(400,17) combinations at p=7, **prove structurally that any 17-vector set spanning F_p^4 is forced by ZSF constraints into doubled+e1 form up to GL conjugacy.** Since any spanning set can be coordinatized to have 4 basis vectors, the remaining 13 are heavily constrained—the algebraic argument might be tighter than computational search.
